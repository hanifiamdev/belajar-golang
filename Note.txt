## Running file go ##
- perintah untuk build
  go build namafile.go
- perintah untuk eksekusi
  ./namafile.go
- hasil file akan mengikuti sistem operasi saat file .go di compile
- Jika ingin menjalankan tanpa kompilasi karena masih tahap belajar / test kode saja bisa gunakan perintah 
  go run namafile.go
